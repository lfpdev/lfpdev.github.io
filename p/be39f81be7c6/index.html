文生图-Midjourney

发表于 2024-07-21 分类于 AIGC

简介

介绍如何通过 Midjourney 进行文生图，包括提示词、辅助工具等

AI绘画的主要应用场景

  • 配图：文字内容的配图
  • 真实照片+换装（重新打光）：给真实照片上的人物换装，然后再重新打光
  • 图片按比例放大、横向、纵向的扩展
  • 生成需要商业授权的人物
  • 自有IP的形象生成不同角度的图片

不适合真实场景

如何分辨是否AI生成

  • 看光影是否符合实际物理规律
  • 看人物的手

Midjourney

特点

  1. 操作简单，可控制性较差

    简单提示词就可以生成质量较好的图，例如：落日，大漠

  2. 图片创意较好，真实性较差

应用场景

  1. 适合创作，通过一句提示词生成具有创意的图片（文生图）
  2. 不适合通过提示词对图片进行局部进行修改（图生图方式）

工作方式：提示词不变，只改变参数，每次生成多个图，从大量的图中挑选满意的图片

如何使用

  • 官方网站

    生成图片：通过 discord，新建服务器，邀请midjourney机器人，通过/imagine 命令输入 prompt

    邀请midjourney机器人步骤：添加首个APP->来看看吧->搜索 midjourney -> 添加APP -> 授权

    其他命令

    • /settings 设置 midjourney
    • /describe + MJ生成的图片URL ，获取图片 prompt
    • /info 当前账号的信息

  • 中文合作网站

    生成图片：

    • 左侧选择模型，输入框直接输入 prompt 即可

    • 输出图片的操作方式

      • u1-u4

        查看所选图像的添加更多细节后的图像

        下载、进行局部重绘，无损放大，调整创作等

      • v1-v4

        生成和所选图像有细微变化的新图像

    • 迭代

      • 重新生成

      • 优化提示词

        提示词少，MJ发挥的空间会更大

    教程

  • 官方API

提示词

万能公式

主体描述+环境场景+艺术风格+媒介材料+摄像机视角+精度定义

可简化为：主体+环境+个性化

  • 如果是3D渲染，再加上3D软件+渲染器+材质+光照+逼真的渲染风格

主体描述

一句话告诉AI你要描述的画面，结构：修饰词+主体+场景+动作

例如：一个美丽的日本女人英格丽长背心站在白墙旁

  • 可描述的一切确切的符合物理规律的动物、事物都可以作为主体

    不符合物理规律的很难生成，例如：苹果形状的水珠

  • 主体的材质可以是虚拟的，例如：蓝色的冰构成的树叶

环境场景

你想象的主体所处的环境，光线、氛围、色系

==个性化==
艺术风格

设计风格/艺术家名字/漫画影视作品/艺术网站

媒介材料

油画/水彩/摄影/手稿/雕塑/陶瓷/布料/黏土/石头

摄影机视角

特写视图/两点透视/广角镜头/景深/正视图/全身照

精度定义

高品质/超级细节/复杂细节/高分辨率/HD/2K/4K/尺寸比例

示例

  • knolling 风格

广告行业
情景描述

不需要事物的细节，只描述情景，AI自动生成符合情景的事物细节

技巧

AI生成

让AI参考优秀的提示词生成

规避政策

  • 避免人物作者的名字，用书名、电影名、出版社名替代

参数

参考：https://www.midjourny.cn/tutorial/15.html

在提示词后添加参数，参数和参数值之间、多个参数之间用空格分隔

  • 通过控制变量法，一步步修改参数
–aspect –ar

调整图片的宽高比例，默认宽高比为 1:1

要结合提示词修改，例如一个靠墙站着拍照的美女，图片比例就需要是纵向的

–seed

随机数种子，参数值为一个0到4294967295之间的数字，重绘时使用

–no

降低元素出现的概率

输入中文提示词时，会自动翻译为英文，需要通过权重来表示

权重：控制不同元素在生成图像时的影响程度

  • 格式：词语::权重值
  • 可选值：默认为1
    • -1：出现几率小，类似 --no
    • 2：出现几率大
–quality –q

调节渲染质量及耗时

可选值：0.25、0.5、1、2，默认为 1。数字越低，生成图像的细节越少，速度越快

在画一些密集性的东西，例如城中村楼房时，将值调大

细节跟图片质量是两个不同的概念，细节少，图片质量不一定差

–chaos –c

调节结果的多样性

取值范围是 0 至 100，默认0，越小相似度越高

一组事物中，不同元素之间的相似度，例如货架上的一排玩偶，值越大，相互之间的区别就越大

–stylize

调整风格化程度

取值范围是0到1000，默认100。数值大小表示在生成图像时施加的风格化程度，具体来说就是创意和艺术加工的强度

类似 chatgpt 的 temperature

要结合有关风格的提示词使用

  • 值越大，越抽象，更有创意
  • 值越小，越写实，更真实

结合 --style raw 使用的场景：整体上保持接近描述内容的“原始性”，但又希望在某些细节或某个特定层面上保留一些艺术效果

–niji

切换 niji 模型，niji模型擅长创作卡通或动画风格的图像

  • niji 5 细节模糊一点，更复古、沧桑
  • niji 6 细节更清晰一点，更写实、像宫崎骏画风
–style raw

倾向于输出较为“原始”的、接近描述内容的结果

–v

调整 MJ 模型，默认 6

  • 5、5.2：低饱和度，具有艺术风格
  • 6：更真实，细节更好
–iw

图生图模式，新图和原图的相似度

–sref

风格参考、风格迁移

参考指定URL的风格

–sw

参考的比例，取值范围：0到1000，默认100

  • 值越小越柔和，0：关闭
  • 值越大细节越多，画面锐度越高
–cref

角色参考、角色迁移

参考指定URL的角色

–cw

参考的比例，取值范围：0到1000，默认100

  • 0：只有面部迁移（适合修改衣服、发型的情况下使用）
  • 100:衣服、面容、发型、服装穿搭全部迁移，只改变背景

格式：--cref（空格）1或多个图片链接（空格）--cw（空格）0-100
