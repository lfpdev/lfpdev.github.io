picgo

发表于 2021-10-15 分类于 Tool

简介

一个用于快速上传图片并获取图片 URL 链接的工具

官网：https://picgo.github.io/PicGo-Doc/

安装

  1. 下载 AppImage 格式文件

    https://github.com/Molunerfinn/PicGo/releases/

    如果安装了 AppImageLauncher，直接丢进 Applications 文件夹即可，省去下面的步骤

  2. 添加可执行权限

  3. 提取 AppImage 文件

    执行完之后会出现一个文件夹 squashfs-root

  4. 在主目录中创建 opt/PicGo/ 目录，将 squashfs-root 移动到这里

  5. 添加 desktop 文件

    1. 进入 squashfs-root 目录

    2. 将 picgo.desktop 文件中 Exec 和 Icon 字段的值改为绝对路径

      注意图片后面要加上.png

    3. 为 picgo.desktop 赋予可执行权限

    4. 将 picgo.desktop 复制到 /usr/share/applications/ 目录下

    5. 此时在应用程序列表中就有Picgo的图标了

  6. 安装 xclip 软件「可选」

    使用剪贴板图片上传功能，需要用到 xclip 软件

图床

阿里云oss

  • 购买对象存储OSS资源包

    标准（LRS）存储包即可

  • 创建 bucket

  • 创建 AccessKey

    1. 创建RAM子帐号

    2. 添加AliyunOSSFullAccess权限

    3. 创建 AccessKey

图床设置

阿里云oss

  • 设定存储空间名
    需要与创建的 bucket 名一致
  • 指定存储路径
    如果bucket里面创建了目录，则这里添加目录路径

Picgo设置

选择 PicGo 设置 ，个性化设置

  • 选择显示的图床
  • 打开
    1. 时间戳重命名
    2. 上传后自动复制URL
  • 关闭
    1. 上传前重命名
