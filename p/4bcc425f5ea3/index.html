发表于 2024-06-30 分类于 AIGC

简介

介绍如何通过 Ollama 在 Colab 上部署大模型

  1. 注册 Colab
  2. 注册 ngrok

步骤

  1. 新建 notebook

    File->new notebook in Drive

  2. 修改运行时类型

    Connect-> Change runtime type

    • 运行时类型选择 Python3

    • 硬件加速器选择 T4GPU

    连接->连接到托管的运行时：T4

  3. 在新的 notebook 中添加两段 Code，点击最侧的执行按钮

    第一段

    第二段

  4. 看到 starting ngrok http 11434 –host-header localhost:11434 等日志，表示部署成功

  5. 在 ngrok 官网 endpoints 查看公网URL，点击 URL->view site，显示 Ollama is running，表示代理成功

  6. 在本地修改 OLLAMA_HOST 环境变量为刚才的公网URL：https://4xxb-3x-1xx-1x1-1x4.ngrok-free.app

    此时在本地客户端输入 ollama 的命令会在 colab 上执行
