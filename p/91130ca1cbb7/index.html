第三方支付

发表于 2022-03-28 分类于 Payment

消费者购买商家的东西，通过第三方平台支付，第三方支付对接各种资金清算机构（如各大银行的支付接口），资金经清算机构清算后转移到收款人账户

第三方含义

业务流程

消费者和商家之间的中介

金融行业

非金融机构支付服务，是指非金融机构在收付款人之间作为中介机构提供下列部分或全部货币资金转移服务

  1. 网络支付；

    指依托公共网络或专用网络在收付款人之间转移货币资金的行为，包括货币汇兑、互联网支付、移动电话支付、固定电话支付、数字电视支付等。

  2. 预付卡的发行与受理；

    指以营利为目的发行的、在发行机构之外购买商品或服务的预付价值，包括采取磁条、芯片等技术以卡片、密码等形式发行的预付卡。

    • 京东E卡

      京东的单用途商业预付卡，仅能购买京东自营商品

  3. 银行卡收单；

    指通过销售点（POS）终端等为银行卡特约商户代收货币资金的行为。

  4. 中国人民银行确定的其他支付服务。

作用

  • 银行系统安全等级要求高，支付体验差。第三方支付平台整合各大银行的支付接口，提供方便快捷的支付方式

应用领域

  • 电子商务

  • 跨境支付

  • 在线支付

支付平台

国外

  • 欧美国家的银行不会向第三方支付机构开放自己的账户资源，即便是PayPal，在客户使用其进行付款时，也要向Visa和万事达卡支付费用，同时还要受到严格的监管

Paypal

Stripe

Braintree

国内

  • 需要第三方支付牌照（支付业务许可证）
  • 中国第三方支付市场的兴起和壮大，完全得益于中国的金融体系，以及央行支付系统、银联网联清算系统等一系列金融基础设施的建立，各级商业银行承担了账户安全、实名验证、手机绑卡等繁琐的基础工作。如果没有这些体系的支持，第三方支付完全就是无源之水、无本之木。
  • 支付宝和微信支付，并不完全以支付本身来盈利，而是借助支付工具形成资金池，从而沉淀大量的备付金，再通过类似余额宝、花呗等产品来盈利
  • 国内第三方平台有：支付宝、微信支付、银联云闪付、电信翼支付、百度钱包、京东金融、小米金融等

支付宝

  • ALL-IN-ONE收单模式
  • 账户余额转入货币基金（余额宝）
  • 用户保证体系
  • 二维码支付
  • 芝麻信用分、蚂蚁森林、花呗、借呗等产品
  • 2015年，支付宝向生活服务平台转型，陆续打通生活、金融、消费、理财等多个场景
  • 2017年，蚂蚁宣布未来只做tech（技术），支持金融机构做好fin（金融），由Fintech变成Techfin，为金融机构提供技术、数据分析支撑。金融科技服务

微信支付

  • 2017年9月，对标“余额宝”的“零钱通”上线；
  • 2018年1月，“信用分”（现在更名为“支付分”）开始公测，显然这是微信版的“芝麻信用分”；
  • 2020年3月，消费信贷产品“分付”开始测试，其对标的当然是“花呗”。

国付宝

  • 成立于2011年1月，是一家以第三方支付为基础的科技金融综合服务平台，旗下拥有针对 B2B/B2C的网络支付产品、移动支付产品、金融服务产品等，针对跨境商贸领域的跨境支付解决方案等。

  • PayPal全资控股的美银宝信息技术（上海）有限公司间接100%控股国付宝

跨境支付

进入中国市场不一定会将重心放在中国境内支付，而是会发挥其在跨境支付方面的优势，盯住Visa、万事达卡、美国运通这些国际卡组织进入中国清算市场后形成的“商机“。其服务的内容包括

  1. 国外用户在中国境内的消费提供支付业务；
  2. 国内个人及企业以后在国际上的跨境支付业务

美国运通已经于2020年6月获得中国银行卡清算牌照，万事达卡与网联的合资企业也即将进入到获批筹备阶段
